# USP_ALGEBRA_LINEAR

## CONTEXTUALIZAÇÃO
Este repositório reúne os códigos e materiais desenvolvidos no contexto da disciplina de Álgebra Linear. O conteúdo foi criado para apoiar o aprendizado prático dos conceitos apresentados em aula e está organizado em três blocos principais: Exemplos, Exercícios e o Projeto final da disciplina.

Os arquivos presentes são, em sua maioria, notebooks Jupyter (.ipynb) e recursos auxiliares (imagens, dados e scripts) utilizados para exploração, experimentação e resolução de problemas numéricos e simbólicos relacionados à Álgebra Linear.

## EXECUÇÃO
Os notebooks e scripts aqui contidos foram desenvolvidos como material de estudo e prática. 

Para executar e visualizar os notebooks localmente, recomenda-se criar um ambiente Python e instalar o Jupyter. 

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install jupyter numpy scipy matplotlib
```

## ESTRUTURAÇÃO DO REPOSITÓRIO
	* **Examples/**
		- Contém notebooks e demonstrações que ilustram conceitos e métodos da disciplina (SVD, mínimos quadrados).
	* **Exercises/**
		- Coleção de exercícios resolvidos e/ou propostos durante as aulas práticas. Em geral são notebooks com enunciados, soluções e visualizações.
	* **Project/**
		- Trabalho final da disciplina, contendo um notebook explicativo sobre a Transformação Rápida de Fourier, como uma forma de otimizar a convolução de uma imagem para aplicação do efeito de blur.
